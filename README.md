# Student Performance Predictor

A machine learning model that predicts a student's final score based on study habits and academic history.

## Results
- **R2 Score: 0.83 (83% accuracy)**
- **Mean Error: only 4.31 marks**
- Algorithm: Random Forest Regressor
- Dataset: 200 students, 4 features

## What I built
- Generated realistic student dataset using numpy
- Trained a Random Forest regression model
- Visualized which factors affect performance most

## Key Finding
Previous academic score and study hours are the strongest predictors of student performance.
